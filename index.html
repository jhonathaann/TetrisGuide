Um Guia de Combos no Tetris

Conceitos Básicos

Antes de começarmos com as jogadas especiais, vamos revisar os conceitos básicos do jogo. Tetris é um jogo de quebra cabeça, onde seu objetivo é empilhar tetraminós que descem pela tela e assim formam linhas horizontais no "chão". Quando uma linha é feita, ela desintegra e as peças superiores descem, e o jogador ganha pontos.

Tetraminós:

Ao todo, existem 7 peças no jogo, cada uma com características diferentes.

Combos Simples

Nesta seção, vamos aprender duas jogadas especiais bem simples, e que não exigem muita prática para serem realizadas.

Sequencial Lines:

O segredo para esse combo é: não tem segredo... A ideia aqui é bem simples, quantos mais linhas você faz de forma consecutiva, maior a quantidade de pontos conquistados (E se estivermos falando de PVP, maior a quantidade de linhas enviados para o seu adversário.)

Combo de 3/4 linhas:

Para essa jogada, o que você precisa fazer é preparar o terreno deixando apenas uma coluna vazia e, nesse espaço que ficou você posiciona a peça I(peça azul de maior comprimento).

Dica: Pratique o posicionamento da peça I para maximizar o número de linhas limpas de uma vez.
